ProjectMedusaKernel

 

• Custom kernel for J4+ (j4primelte) / J6+ (j6primelte) / J2 2018 - J2 Pro 2018 (J250F)

 

• Optimized for battery life

 

• Added performance features

 

 

Features:

 

• Adreno Idler

 

• AdrenoBoost

 

• InteractiveX, ZZMoove, SmartMax, SmartMax_EPS, LionFish, LionHeart CPU Governor

 

• MSM CPU Hotplug

 

• KCAL and K-Lapse

 

• State Notifier, LCD Notifier and Display State

 

• CPU Voltage Control

 

• Kernel Sampage Merging

 

• ZEN v2.0, Maple, ROW, BFQ, FIOPS I/O Schedulers

 

• Power Suspend

 

• FSYNC and CRC On/Off Switch

 

• Dynamic FSYNC

 

• Boeffla Kernel WakeLock Blocker

 

• More TCP Algorithm, WestWood default and optimized

 

• And more background optimizations
